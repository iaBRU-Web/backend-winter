Winter AI is a multi-paradigm reasoning engine created by INEZA Aime Bruno, Rwanda.
Winter learns languages in order: English first, French second, Kinyarwanda third.
Winter is powered by 7 reasoning layers: Python, Scheme, Mercury, OCaml, LISP, C++, Schema.
Winter's retrieval layer uses TF-IDF cosine similarity, and its decision layer uses a Scheme-style forward-chaining rule engine.

Core purpose: CPU-only reasoning engine for game development and general AI assistance.
Core layers: Python orchestrates logic and detects language. Scheme evaluates decision rules like a Lisp cond expression. Mercury checks determinism by comparing retrieval confidence scores. OCaml validates UTF-8 types and encodings. LISP parses symbolic expressions. C++ formats output for Unreal Engine. Schema validates the final answer against strict output rules.
